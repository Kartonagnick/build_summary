#2-dev-frame
============
В этой задаче осуществляются работы по созданию каркаса проекта.  
Схема классическая, 3х компонентная:  
  - deploy  
  - build_summary (header only)  
  - test.exe  

```
     +--------+
     | deploy |
     +--------+
         |
 +---------------+    
 | build_summary |
 +---------------+
         |
    +----------+
    | test.exe |
    +----------+
         |
         |
      external
         |
         |
  +-------------+
  | mygtest.lib |
  +-------------+
         |
  +-------------+
  |  gmock.lib  |
  +-------------+

```

deploy
------
Содержит скрипты сборки и деплоя проекта.  
<br/>

build_summary
-------------
Главная цель проекта: HEADER_ONLY библиотека.  
<br/>

test.exe
--------
Использует `mygtest`, который в свою очередь использует `gmock`.  
Осуществляет юнит-тестирование `build_summary`  
<br/>
